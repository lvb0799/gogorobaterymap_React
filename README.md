GOGORO Battery Map (React)
===

    練習：運用React 搭配React leaflet 抓取GOGORO的充電站位置API

    之後有空再加上React-redux或是Hook

## Installation
* **git clone project**
* **use VSCode and open to project folder**
* **install npm package**

  ```
  npm install
  ```
* **run NPM or YARN**
  ```
  npm start
  or
  yarn start
  ```
